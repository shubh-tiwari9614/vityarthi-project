VIT Bhopal Digital Library System — Problem Statement

This project aims to develop a simple Digital Library Management System using Python. The system allows students to view available books, issue books, and return issued books with an automatic fine calculation for late returns. The program uses loops, conditions, and dictionaries as required in the assignment.


---

Scope of the Project

The scope of this project includes:

Displaying a list of available books.

Allowing users to issue a book by entering the date (day number).

Recording issue details using a dictionary (issued_books = {"Book Name": issue_day}).

Allowing users to return books and calculating:

Number of days used

Whether the return is on time

If late, the fine amount (₹10 per extra day)


Updating the available books list dynamically.

Providing a menu-driven interface using loops and conditions.

Allowing the user to exit the system smoothly.


The project does not include a database, GUI, or online connectivity. It is a basic Python console program focusing on logic building and Python fundamentals.
